Enregistrement de domaine associé
Si vous avez déjà enregistré un domaine manuellement avec un registraire qui a été ajouté sur la page DNS Domain Registrars, vous pouvez utiliser ce formulaire pour informer Virtualmin de l'enregistrement. Cela lui permet de désenregistrer le domaine si vous supprimez le serveur virtuel ou d'essayer de modifier l'enregistrement si le nom de domaine est modifié.

Pour utiliser le formulaire, sélectionnez d'abord le compte que vous avez utilisé dans le menu Enregistré sous compte . Si vous disposez d'un ID ou d'un code d'enregistrement pour le domaine, saisissez-le dans le champ ID with registrar . Enfin, si vous souhaitez que les serveurs de noms définis sur le bureau d'enregistrement soient mis à jour pour correspondre à ce système Virtualmin, définissez Mettre à jour les serveurs de noms sur Oui .

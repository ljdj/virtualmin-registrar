Pindahkan Pendaftaran Domain
Sekiranya domain ini sedang didaftarkan dengan pendaftar lain, anda boleh meminta domain tersebut dipindahkan ke salah satu akaun pendaftar anda di halaman ini. Pemindahan tidak berlaku seketika - mungkin memerlukan beberapa hari, dan memerlukan proses manual dengan pendaftar lama.

Setelah pemindahan selesai, anda boleh menggunakan halaman Import Domain Pendaftaran untuk membawa domain di bawah kawalan Virtualmin.

Assoziierte Domain-Registrierung
Wenn Sie eine Domain bereits manuell bei einem Registrar registriert haben, der auf der Seite DNS-Domain-Registrare hinzugefügt wurde, können Sie Virtualmin über dieses Formular über die Registrierung informieren. Auf diese Weise kann die Registrierung der Domäne aufgehoben werden, wenn Sie den virtuellen Server löschen, oder es kann versucht werden, die Registrierung zu ändern, wenn der Domänenname geändert wird.

Um das Formular zu verwenden, wählen Sie zunächst das von Ihnen verwendete Konto im Menü Registriert unter Konto aus . Wenn Sie eine ID oder einen Registrierungscode für die Domain haben, geben Sie diese in das Feld ID mit Registrar ein. Wenn Sie möchten, dass die beim Registrar festgelegten Nameserver aktualisiert werden, um mit diesem Virtualmin-System übereinzustimmen, ändern Sie Nameserver aktualisieren in Ja .

Domain-Registrierung erneuern
Wenn eine Domain-Registrierung kurz vor dem Ablauf steht, können Sie sie mit diesem Formular erneuern. Geben Sie einfach die Anzahl der Jahre ein, in denen das Formular erneuert werden soll, und klicken Sie auf Domain jetzt erneuern. Das Formular wird dann mit dem Konto neu registriert, mit dem es ursprünglich erstellt wurde. Ihrem Konto beim Registrar wird höchstwahrscheinlich eine angemessene Gebühr für die Verlängerung berechnet.

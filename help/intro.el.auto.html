Εγγραφή τομέα DNS
Η προσθήκη DNS Domain Registration σάς επιτρέπει να αυτοματοποιήσετε τη διαδικασία καταχώρησης τομέων από το Virtualmin, αντί να χρειάζεται να τα δημιουργήσετε χειροκίνητα. Για να γίνει αυτό, πρέπει να γνωρίζει τις λεπτομέρειες των λογαριασμών που έχετε σε έναν ή περισσότερους καταχωρητές, προκειμένου να καλέσετε τα API τους για την εκτέλεση της πραγματικής δημιουργίας τομέα.

Σε αυτήν τη σελίδα παρατίθενται οι λογαριασμοί καταχωρητή DNS που έχετε προσθέσει στο Virtualmin για χρήση κατά την εγγραφή τομέα. Για να προσθέσετε έναν νέο καταχωρητή, επιλέξτε την εταιρεία από το μενού Προσθήκη υπάρχοντος λογαριασμού στο κάτω μέρος της σελίδας και κάντε κλικ στο κουμπί Έναρξη προσθήκης . Πρέπει να έχετε τουλάχιστον έναν λογαριασμό στη λίστα και να είναι ενεργός για να μπορέσετε να εγγραφείτε τομείς στο Virtualmin.

Οι λογαριασμοί μπορούν επίσης να ενεργοποιηθούν ή να απενεργοποιηθούν επιλέγοντας τους χρησιμοποιώντας τα πλαίσια ελέγχου στα αριστερά του πίνακα και κάνοντας κλικ στο κατάλληλο κουμπί. Ένας λογαριασμός καταχωρητή που έχει επισημανθεί ως απενεργοποιημένος δεν θα χρησιμοποιηθεί κατά την εγγραφή νέων τομέων.

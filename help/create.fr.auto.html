Créer un compte de registraire
Cette page vous permet de créer un nouveau compte avec un registraire, qui sera facturé sur votre carte de crédit ou solde prépayé. Les détails exacts que vous devez saisir différeront selon les bureaux d'enregistrement, mais en général, vous devrez sélectionner un nom d'utilisateur et un mot de passe, entrer les coordonnées et les informations de paiement par carte de crédit.

Une fois le compte créé, il peut s'écouler plusieurs heures avant de pouvoir être utilisé pour enregistrer des domaines DNS pour des serveurs virtuels. Si la création échoue pour une raison quelconque (comme un numéro de carte de crédit non valide), un message d'erreur s'affichera cependant immédiatement.

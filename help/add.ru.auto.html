Добавить учетную запись регистратора
На этой странице вы можете сообщить Virtualmin об учетной записи регистратора, которую вы уже настроили, чтобы ее можно было использовать для регистрации новых доменов. Наиболее важные поля в форме:
  • Описание аккаунта
    Краткое описание для идентификации этого аккаунта.
  • Идентификатор учетной записи и пароль учетной записи
    Данные аутентификации аккаунта предоставлены вашим регистратором. Они будут проверены при нажатии кнопки « Создать» .
  • Домены верхнего уровня, поддерживаемые регистратором
    Не редактируемое поле, а список доменов верхнего уровня, для которых этот регистратор может использоваться.
  • Дополнительно ограничить доменами верхнего уровня
    Если вы хотите ограничить эту учетную запись регистратора созданием доменов только под подмножеством возможных TLD, введите их здесь. Это может быть полезно, если у вас есть несколько учетных записей, и некоторые предлагают лучшие цены, чем другие на одном домене.
При нажатии кнопки « Создать » данные регистратора будут проверены, а затем учетная запись добавлена в список используемых регистраторов Virtualmin.

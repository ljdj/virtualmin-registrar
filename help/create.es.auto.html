Crear cuenta de registrador
Esta página le permite crear una nueva cuenta con un registrador, que se facturará a su tarjeta de crédito o saldo prepago. Los detalles exactos que debe ingresar diferirán entre los registradores, pero en general deberá seleccionar un nombre de usuario y contraseña, ingresar los datos de contacto y la información de pago con tarjeta de crédito.

Una vez que se crea la cuenta, puede haber un retraso de varias horas antes de que pueda usarse para registrar dominios DNS para servidores virtuales. Si la creación falla por algún motivo (como un número de tarjeta de crédito no válido), se mostrará un mensaje de error inmediatamente.

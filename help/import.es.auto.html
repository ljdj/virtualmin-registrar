Registro de dominio asociado
Si ya ha registrado un dominio manualmente con un registrador que se ha agregado en la página Registradores de dominio DNS, puede usar este formulario para informar a Virtualmin sobre el registro. Esto le permite cancelar el registro del dominio si elimina el servidor virtual, o intentar modificar el registro si se cambia el nombre del dominio.

Para usar el formulario, primero seleccione la cuenta que usó en el menú Registrado en cuenta . Si tiene una ID o código de registro para el dominio, ingréselo en el campo ID con registrador . Finalmente, si desea que los servidores de nombres configurados en el registrador se actualicen para que coincidan con este sistema Virtualmin, cambie Actualizar servidores de nombres a Sí .

Registro de domínio associado
Se você já registrou um domínio manualmente com um registrador que foi adicionado na página Registradores de domínio DNS, você pode usar este formulário para informar o Virtualmin sobre o registro. Isso permite cancelar o registro do domínio se você excluir o servidor virtual ou tentar modificar o registro se o nome do domínio for alterado.

Para usar o formulário, primeiro selecione a conta que você usou no menu Registrado em conta . Se você tiver um ID ou código de registro para o domínio, digite-o no campo ID com registrador . Por fim, se você deseja que os servidores de nomes definidos no registrador sejam atualizados para corresponderem ao sistema Virtualmin, altere Atualizar servidores de nomes para Sim .

Adicionar conta de registrador
Esta página permite que você informe ao Virtualmin sobre uma conta de registrador que você já configurou, para que possa ser usada para registrar novos domínios. Os campos mais importantes do formulário são:
  • Descrição da conta
    Uma breve descrição para identificar esta conta.
  • ID de login da conta e senha da conta
    Os detalhes de autenticação da conta fornecidos pelo seu registrador. Eles serão verificados quando o botão Criar for clicado.
  • Domínios de nível superior suportados pelo registrador
    Não é um campo editável, mas uma lista de TLDs para os quais este registrador pode ser usado.
  • Além disso, limite para domínios de nível superior
    Se você deseja limitar essa conta de registrador à criação de domínios em apenas um subconjunto dos possíveis TLDs, insira-os aqui. Isso pode ser útil se você tiver várias contas e algumas oferecerem preços melhores que outras no mesmo domínio.
Quando o botão Criar é clicado, os detalhes do registrador serão validados e a conta adicionada à lista de registradores utilizáveis do Virtualmin.

Автоматическое продление домена
Чтобы предотвратить истечение срока регистрации доменов по истечении срока их действия, Virtualmin может автоматически продлить их для вас, когда они приближаются к дате истечения срока действия. Чтобы включить эту функцию, измените Автоматическое продление доменов, близких к истечению срока действия? в поле Да и введите количество дней до истечения срока действия каждого домена, которое необходимо продлить. Рекомендуется что-то в районе 30 дней, чтобы у вас было достаточно времени, чтобы среагировать в случае неудачи обновления.

Период, на который продлевается каждый домен, можно установить с помощью поля Годы для продления . По умолчанию будет использоваться начальный период регистрации, установленный для этой учетной записи.

Если в поле « Отправить уведомление о продлении» указан адрес электронной почты, ему будет отправлен отчет со списком всех доменов, обновляемых каждый день. Любые домены, для которых продление не удалось из-за ошибки регистратора, также будут включены.

Registrazione domini associati
Se hai già registrato manualmente un dominio con un registrar che è stato aggiunto nella pagina Registrar del dominio DNS, puoi utilizzare questo modulo per comunicare a Virtualmin della registrazione. Ciò consente di annullare la registrazione del dominio se si elimina il server virtuale o di tentare di modificare la registrazione se si modifica il nome del dominio.

Per utilizzare il modulo, selezionare innanzitutto l'account utilizzato dal menu Account registrato . Se hai un ID o un codice di registrazione per il dominio, inseriscilo nel campo ID con registrar . Infine, se si desidera che i nameserver impostati nel registrar vengano aggiornati in modo che corrispondano a questo sistema Virtualmin, modificare Aggiorna nameserver su Sì .

DNS-domänregistrering
DNS-domänregistreringspluginet låter dig automatisera processen för att registrera domäner från Virtualmin, snarare än att behöva skapa dem manuellt. För att göra detta måste den känna till detaljerna för konton som du har hos en eller flera registratorer, för att ringa deras API: er för att utföra själva domänskapandet.

Denna sida visar DNS-registratorkonton som du har lagt till Virtualmin för användning vid registrering av domän. För att lägga till en ny registrator väljer du företaget från Lägg till ett befintligt konto- meny längst ner på sidan och klickar på Start Lägg till- knappen. Du måste ha minst ett konto listat och aktivt innan du kan registrera domäner med Virtualmin.

Konton kan också aktiveras eller inaktiveras genom att markera dem med kryssrutorna till vänster om tabellen och klicka på lämplig knapp. Ett registrarkonto som är markerat som inaktiverat kommer inte att användas när du registrerar nya domäner.

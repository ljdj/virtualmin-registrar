DNS 도메인 등록
DNS 도메인 등록 플러그인을 사용하면 Virtualmin 내에서 도메인을 수동으로 만들지 않고 도메인을 등록하는 프로세스를 자동화 할 수 있습니다. 이를 위해서는 실제 도메인 생성을 수행하기 위해 API를 호출하기 위해 하나 이상의 레지스트라가있는 계정의 세부 사항을 알아야합니다.

이 페이지에는 도메인 등록시 사용하기 위해 Virtualmin에 추가 한 DNS 등록자 계정이 나열되어 있습니다. 새 등록자를 추가하려면 페이지 하단의 기존 계정 추가 메뉴에서 회사를 선택하고 추가 시작 버튼을 클릭하십시오. Virtualmin에 도메인을 등록하려면 먼저 하나 이상의 계정이 나열되고 활성화되어 있어야합니다.

표 왼쪽의 확인란을 사용하여 계정을 선택하고 해당 버튼을 클릭하여 계정을 활성화하거나 비활성화 할 수도 있습니다. 비활성화 된 것으로 표시된 등록자 계정은 새 도메인을 등록 할 때 사용되지 않습니다.

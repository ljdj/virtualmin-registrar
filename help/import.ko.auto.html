준회원 도메인 등록
DNS Domain Registrars (DNS 도메인 등록 기관) 페이지에 추가 된 등록 기관으로 도메인을 수동으로 등록한 경우이 양식을 사용하여 Virtualmin에 등록에 대해 알릴 수 있습니다. 따라서 가상 서버를 삭제하면 도메인 등록을 해제하거나 도메인 이름이 변경된 경우 등록을 수정할 수 있습니다.

양식을 사용하려면 먼저 계정 아래 의 등록 메뉴에서 사용한 계정을 선택하십시오. 도메인의 ID 또는 등록 코드가있는 경우 등록자 ID 필드에 입력하십시오. 당신이 원한다면 마지막으로, 등록 기관에서 설정 한 네임 서버는이 Virtualmin 시스템, 예로 변경 업데이트 네임 서버 일치하도록 업데이트합니다.

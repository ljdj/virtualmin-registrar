Renovação automática de domínio
Para impedir que os registros de domínio expirem quando expiram, o Virtualmin pode renová-los automaticamente quando você se aproximar da data de validade. Para habilitar esse recurso, altere Renovar automaticamente domínios próximo do vencimento? campo para Sim e insira o número de dias antes da expiração em que cada domínio deve ser renovado. Recomenda-se algo na área de 30 dias, para que você tenha tempo de sobra para reagir se a renovação falhar.

O período durante o qual cada domínio é renovado pode ser definido usando o campo Anos para renovar para . Por padrão, o período de registro inicial definido para esta conta será usado.

Se um endereço de e-mail for inserido no campo Enviar notificação de renovação para , um relatório listando todos os domínios renovados a cada dia será enviado a ele. Todos os domínios cuja renovação falhou devido a um erro do registrador também serão incluídos.

Ajouter un compte de registraire
Cette page vous permet d'indiquer à Virtualmin un compte de registraire que vous avez déjà configuré, afin qu'il puisse être utilisé pour enregistrer de nouveaux domaines. Les champs les plus importants du formulaire sont:
  • Description du compte
    Une brève description pour identifier ce compte.
  • Identifiant de connexion au compte et mot de passe du compte
    Les détails d'authentification du compte fournis par votre registraire. Ceux-ci seront vérifiés lorsque vous cliquez sur le bouton Créer .
  • Domaines de premier niveau pris en charge par le registraire
    Pas un champ modifiable, mais une liste de TLD pour lesquels ce bureau d'enregistrement peut être utilisé.
  • Limite supplémentaire aux domaines de premier niveau
    Si vous souhaitez limiter ce compte de bureau d'enregistrement à la création de domaines sous seulement un sous-ensemble des TLD possibles, entrez-les ici. Cela peut être utile si vous avez plusieurs comptes et que certains offrent de meilleurs prix que d'autres sur le même domaine.
Lorsque le bouton Créer est cliqué, les détails du registraire seront validés, puis le compte ajouté à la liste des registraires utilisables de Virtualmin.

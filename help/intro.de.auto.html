DNS-Domänenregistrierung
Mit dem DNS Domain Registration Plugin können Sie den Prozess der Registrierung von Domains in Virtualmin automatisieren, anstatt sie manuell erstellen zu müssen. Dazu müssen die Details der Konten bekannt sein, die Sie bei einem oder mehreren Registraren haben, damit deren APIs aufgerufen werden können, um die eigentliche Domänenerstellung durchzuführen.

Auf dieser Seite werden die DNS-Registrierungskonten aufgelistet, die Sie Virtualmin zur Verwendung bei der Registrierung der Domain hinzugefügt haben. Um einen neuen Registrar hinzuzufügen, wählen Sie das Unternehmen im Menü Vorhandenes Konto hinzufügen unten auf der Seite aus und klicken Sie auf die Schaltfläche Hinzufügen hinzufügen. Sie müssen mindestens ein Konto aufgelistet und aktiv haben, bevor Sie Domains bei Virtualmin registrieren können.

Konten können auch aktiviert oder deaktiviert werden, indem Sie sie über die Kontrollkästchen links in der Tabelle auswählen und auf die entsprechende Schaltfläche klicken. Ein Registrar-Konto, das als deaktiviert markiert ist, wird bei der Registrierung neuer Domains nicht verwendet.

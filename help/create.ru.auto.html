Создать учетную запись регистратора
Эта страница позволяет вам создать новую учетную запись с регистратором, который будет выставлен на счет вашей кредитной карты или предоплаченного баланса. Точные данные, которые необходимо ввести, будут отличаться у разных регистраторов, но в целом вам нужно будет выбрать имя пользователя и пароль, ввести контактные данные и информацию об оплате кредитной картой.

После создания учетной записи может пройти несколько часов, прежде чем ее можно будет использовать для регистрации доменов DNS для виртуальных серверов. Если по какой-либо причине создание не удастся (например, неверный номер кредитной карты), сразу же появится сообщение об ошибке.

Registrar-Konto hinzufügen
Auf dieser Seite können Sie Virtualmin über ein Registrar-Konto informieren, das Sie bereits eingerichtet haben, damit damit neue Domains registriert werden können. Die wichtigsten Felder im Formular sind:
  • Account Beschreibung
    Eine kurze Beschreibung zur Identifizierung dieses Kontos.
  • Konto-Login-ID und Konto-Passwort
    Die von Ihrem Registrar bereitgestellten Kontoauthentifizierungsdetails. Diese werden überprüft, wenn Sie auf die Schaltfläche Erstellen klicken.
  • Vom Registrar unterstützte Top-Level-Domains
    Kein bearbeitbares Feld, sondern eine Liste von TLDs, für die dieser Registrar verwendet werden kann.
  • Beschränken Sie sich zusätzlich auf Domänen der obersten Ebene
    Wenn Sie dieses Registrar-Konto darauf beschränken möchten, Domains nur unter einer Teilmenge der möglichen TLDs zu erstellen, geben Sie diese hier ein. Dies kann nützlich sein, wenn Sie mehrere Konten haben und einige bessere Preise bieten als andere auf derselben Domain.
Wenn Sie auf die Schaltfläche Erstellen klicken, werden die Details des Registrars überprüft und das Konto zur Liste der verwendbaren Registrare von Virtualmin hinzugefügt.

การลงทะเบียนโดเมน DNS
ปลั๊กอินการลงทะเบียนโดเมน DNS ช่วยให้คุณสามารถดำเนินการลงทะเบียนโดเมนจากภายใน Virtualmin ได้โดยอัตโนมัติแทนที่จะต้องสร้างด้วยตนเอง ในการดำเนินการนี้จำเป็นต้องทราบรายละเอียดของบัญชีที่คุณมีกับผู้รับจดทะเบียนหนึ่งรายขึ้นไปเพื่อที่จะเรียก API ของพวกเขาเพื่อดำเนินการสร้างโดเมนจริง

หน้านี้แสดงรายการบัญชีผู้รับจดทะเบียน DNS ที่คุณได้เพิ่มไว้ใน Virtualmin เพื่อใช้เมื่อลงทะเบียนโดเมน หากต้องการเพิ่มผู้รับจดทะเบียนรายใหม่ให้เลือก บริษัท จากเมนู เพิ่มบัญชีที่มีอยู่ ที่ด้านล่างของหน้าและคลิกปุ่ม เริ่มเพิ่ม คุณต้องมีบัญชีอย่างน้อยหนึ่งรายการและเปิดใช้งานก่อนที่คุณจะสามารถลงทะเบียนโดเมนกับ Virtualmin

บัญชีสามารถเปิดใช้งานหรือปิดใช้งานได้โดยการเลือกโดยใช้ช่องทำเครื่องหมายทางด้านซ้ายของตารางและคลิกปุ่มที่เหมาะสม บัญชีผู้รับจดทะเบียนที่ทำเครื่องหมายว่าปิดใช้งานจะไม่ถูกใช้เมื่อลงทะเบียนโดเมนใหม่

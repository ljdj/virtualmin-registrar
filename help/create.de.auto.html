Registrar-Konto erstellen
Auf dieser Seite können Sie ein neues Konto bei einem Registrar erstellen, das Ihrer Kreditkarte oder Ihrem vorausbezahlten Guthaben belastet wird. Die genauen Daten, die Sie eingeben müssen, unterscheiden sich zwischen den Registraren. Im Allgemeinen müssen Sie jedoch einen Benutzernamen und ein Passwort auswählen, Kontaktdaten und Kreditkartenzahlungsinformationen eingeben.

Sobald das Konto erstellt wurde, kann es einige Stunden dauern, bis DNS-Domänen für virtuelle Server registriert werden können. Wenn die Erstellung aus irgendeinem Grund fehlschlägt (z. B. aufgrund einer ungültigen Kreditkartennummer), wird sofort eine Fehlermeldung angezeigt.

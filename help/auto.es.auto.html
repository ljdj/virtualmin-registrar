Renovación Automática de Dominio
Para evitar que los registros de dominio caduquen cuando caduquen, Virtualmin puede renovarlos automáticamente cuando se acerquen a la fecha de caducidad. Para habilitar esta función, ¿cambia los dominios de renovación automática cerca del vencimiento? coloque el campo en Sí e ingrese el número de días antes de la fecha de vencimiento que se debe renovar cada dominio. Se recomienda algo en el área de 30 días, para que tenga tiempo de sobra para reaccionar si falla la renovación.

El período para el que se renueva cada dominio se puede establecer utilizando el campo Años para renovar por . Por defecto, se utilizará el período de registro inicial establecido para esta cuenta.

Si se ingresa una dirección de correo electrónico en el campo Enviar notificación de renovación a , se le enviará un informe con todos los dominios renovados cada día. También se incluirán los dominios para los que la renovación falló debido a un error del registrador.

Skapa Registrar konto
På denna sida kan du skapa ett nytt konto hos en registrator, som debiteras ditt kreditkort eller det förbetalda saldot. De exakta uppgifterna du behöver ange skiljer sig mellan registratorer, men i allmänhet måste du välja ett användarnamn och lösenord, ange kontaktinformation och kreditkortsbetalningsinformation.

När kontot har skapats kan det försenas flera timmar innan det kan användas för att registrera DNS-domäner för virtuella servrar. Om skapandet av någon anledning misslyckas (t.ex. ett ogiltigt kreditkortsnummer) visas dock ett felmeddelande omedelbart.

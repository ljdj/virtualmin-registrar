DNS регистрация доменов
Плагин DNS Domain Registration позволяет автоматизировать процесс регистрации доменов из Virtualmin, а не создавать их вручную. Для этого ему нужно знать детали учетных записей, которые у вас есть с одним или несколькими регистраторами, чтобы вызывать их API для выполнения фактического создания домена.

На этой странице перечислены учетные записи DNS-регистраторов, которые вы добавили в Virtualmin для использования при регистрации домена. Чтобы добавить нового регистратора, выберите компанию в меню «Добавить существующую учетную запись» внизу страницы и нажмите кнопку «Начать добавление» . Для регистрации доменов в Virtualmin у вас должна быть хотя бы одна активная учетная запись.

Учетные записи также можно включить или отключить, выбрав их с помощью флажков в левой части таблицы и нажав соответствующую кнопку. Учетная запись регистратора, помеченная как отключенная, не будет использоваться при регистрации новых доменов.

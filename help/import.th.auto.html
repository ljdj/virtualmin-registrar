เชื่อมโยงการลงทะเบียนโดเมน
หากคุณลงทะเบียนโดเมนด้วยตนเองกับผู้รับจดทะเบียนที่เพิ่มไว้ในหน้า DNS Registrars แล้วคุณสามารถใช้แบบฟอร์มนี้เพื่อแจ้ง Virtualmin เกี่ยวกับการลงทะเบียน วิธีนี้ช่วยให้สามารถยกเลิกการลงทะเบียนโดเมนได้หากคุณลบเซิร์ฟเวอร์เสมือนหรือพยายามแก้ไขการลงทะเบียนหากเปลี่ยนชื่อโดเมน

หากต้องการใช้แบบฟอร์มให้เลือกบัญชีที่คุณใช้จากเมนู ลงทะเบียนใต้บัญชี หากคุณมี ID หรือรหัสการลงทะเบียนสำหรับโดเมนให้ป้อนใน ID ด้วย ฟิลด์ ผู้รับจดทะเบียน ท้ายที่สุดถ้าคุณต้องการให้ nameservers ที่ตั้งไว้ที่นายทะเบียนได้รับการปรับปรุงให้ตรงกับระบบ Virtualmin นี้ให้เปลี่ยน Update nameservers เป็น Yes

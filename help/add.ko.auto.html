등록자 계정 추가
이 페이지에서 Virtualmin에 이미 설정 한 등록자 계정에 대해 알려 주어 새 도메인을 등록하는 데 사용할 수 있습니다. 양식에서 가장 중요한 필드는 다음과 같습니다.
  • 계정 설명
    이 계정을 식별하는 간단한 설명입니다.
  • 계정 로그인 ID 및 계정 비밀번호
    등록 기관에서 제공 한 계정 인증 정보 생성 버튼을 클릭하면 확인됩니다.
  • 등록 기관이 지원하는 최상위 도메인
    편집 가능한 필드는 아니지만이 등록자가 사용할 수있는 TLD 목록입니다.
  • 최상위 도메인으로 추가 제한
    이 등록자 계정을 사용 가능한 TLD의 서브 세트만으로 도메인을 작성하도록 제한하려면 여기에 입력하십시오. 이는 여러 계정이 있고 일부가 동일한 도메인에있는 다른 계정보다 더 나은 가격을 제공하는 경우 유용 할 수 있습니다.
만들기 버튼을 클릭하면 등록 기관의 세부 정보가 확인 된 다음 계정이 Virtualmin의 사용 가능한 등록 기관 목록에 추가됩니다.

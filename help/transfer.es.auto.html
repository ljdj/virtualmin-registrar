Transferir registro de dominio
Si este dominio está actualmente registrado con otro registrador, puede solicitar que se transfiera a una de sus cuentas de registrador en esta página. La transferencia no ocurre instantáneamente, puede tomar varios días y requerir procesos manuales con el antiguo registrador.

Una vez que se complete la transferencia, puede usar la página Importar registro de dominio para poner el dominio bajo el control de Virtualmin.
